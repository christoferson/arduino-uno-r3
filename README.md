# Arduino Learning Projects

A collection of Arduino sketches for learning and experimentation with Arduino Uno R3.

## Project Structure

Each project is contained in its own numbered folder with the sketch file matching the folder name.

## Projects

- [001 - Basic LED Blink](001/001.ino) - Blinks the built-in LED on and off at 1-second intervals
- [002 - Button Input](002/002.ino) - TBD
- [003 - PWM Fade](003/003.ino) - TBD


## Hardware Requirements

- Arduino Uno R3 (or compatible board)
- USB cable for programming and power
- Additional components as specified in individual projects

## Getting Started

1. **Install Arduino IDE**
   - Download from [arduino.cc](https://www.arduino.cc/en/software)
   - Install appropriate drivers for your board

2. **Open a Project**
   - Navigate to the desired project folder (e.g., `001/`)
   - Open the `.ino` file in Arduino IDE

3. **Upload to Board**
   - Connect Arduino via USB
   - Select board: **Tools → Board → Arduino Uno**
   - Select port: **Tools → Port → [Your Arduino Port]**
   - Click **Upload** button or press Ctrl+U (Cmd+U on Mac)


## Naming Convention

- Folders are numbered sequentially: `001`, `002`, `003`, etc.
- Sketch files match their folder names: `001.ino`, `002.ino`, etc.
- This is required by Arduino IDE (sketch name must match folder name)
